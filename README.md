# Mini Message Board

A basic Express and EJS-based web application that allows users to submit and view messages in real-time. This project serves as a solution to the Mini Message Board assignment from **The Odin Project** curriculum.

## Description

This application demonstrates the fundamental concepts of routing, middleware, and form handling in Node.js. In this specific implementation, the board is themed around users sharing their favorite tea flavors. 

Users can:
* View a list of all submitted tea recommendations on the index page.
* Access a form to submit their own message (author and tea flavor).
* View individual message details via a dedicated details page.

## Technologies Used

* Node.js
* Express.js
* EJS (Embedded JavaScript templates)
* CSS

## Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server using `npm start` or `node app.js`.
4. Navigate to `http://localhost:3000` in your browser.
